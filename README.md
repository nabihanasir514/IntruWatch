# ![IntruWatch](https://img.shields.io/badge/IntruWatch-Security-blue) IntruWatch

**IntruWatch** is a real-time campus intrusion detection system built with **Python** and **Streamlit**. It monitors live camera feeds, detects unauthorized access using AI-powered face recognition, and alerts the security department.

---

## Features

* Real-time intrusion detection via webcam/camera
* Face recognition for Students, Faculty, and Security Dept
* Guard management using Binary Search Trees (BST)
* Priority alerts and event logging
* Interactive Streamlit dashboard for monitoring

---

## Technologies

* **Python & Streamlit**
* **Libraries:** OpenCV, NumPy, matplotlib, heapq, pickle
* **Data Structures:** Arrays, Linked Lists, BST, Queues, Hash Maps
* **File Handling:** Persistent storage for logs, images, and models

## Installation

```bash
git clone https://github.com/yourusername/IntruWatch.git

streamlit run intruwatch.py
```
